# COVID-19 Trend Analysis & Forecasting

This project analyzes the global spread of COVID-19 and predicts future case trends using data analysis and machine learning techniques in Python.

The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and time-series forecasting.

---

## Project Objective

- Analyze the spread of COVID-19 worldwide
- Identify trends in confirmed, recovered, and death cases
- Visualize infection growth using interactive graphs
- Forecast future COVID-19 cases using time-series models

---

## Dataset

The dataset used in this project contains global COVID-19 statistics, including:

- Country / Region
- Date
- Confirmed cases
- Death cases
- Recovered cases
- Active
- WHO Region

Source dataset: covid_19_clean_complete.csv

The dataset includes daily reports of COVID-19 cases worldwide.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Matplotlib
- Facebook Prophet

## Time Series Forecasting

To predict future COVID-19 cases, a **Facebook Prophet time series model** was used.

Steps:

1. Data preprocessing
2. Time-series modeling
3. Forecast generation
4. Visualization of predicted cases

## Results

Key insights from the analysis:

- Rapid increase in global confirmed cases during the early pandemic stages
- Recovery rates improved significantly over time
- The time-series forecasting model successfully predicted short-term trends in case growth
